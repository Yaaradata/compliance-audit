MVP integration sequencing
For an India MVP with credible coverage of the 22 sample-row controls, recommended sequence:

Wave 1 (weeks 0–8) — CBS event stream; LOS event stream; AML engine; sanctions engine; case management; CKYCR. Covers KYC + Lending + AML deep dives → 14 of the 22 controls.
Wave 2 (weeks 8–16) — NPCI / UPI feed; ITSM; SIEM; FIU-IND outbound; CIMS outbound; CRILC. Adds UPI + IT Ops + reporting outbound → 6 more controls.
Wave 3 (weeks 16–24) — VMO; CMS; HRMS; document management; telephony / ASR. Closes vendor + complaints + audit trail → remaining 2 controls.

Each wave delivers a slice of every persona view (CRO/Compliance/Audit/Ops) — not a single complete persona — because demonstrating drillability across waves is what proves the platform's value proposition.
