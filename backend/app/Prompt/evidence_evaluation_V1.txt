      ┌─ No file uploaded

      │     "Upload the evidence document (PDF, Excel, Word, or image)

      │
      ├─ File uploaded but content is incomplete
      │   → field_feedback[evidence_document] =
      │     a specific one-liner stating exactly what is missing
      │     (e.g. "Include per-flow TLS version and cipher suite

      ├─ File format is unsuitable
