# SWIFT Architecture Diagrams

Place one image per architecture type. Filenames must match:

- **A1.png** — Full Local SWIFT Infrastructure
- **A2.png** — Shared SWIFT Infrastructure (Service Bureau)
- **A3.png** — Connector (Alliance Lite2 / Similar)
- **A4.png** — Customer Connector (Middleware / API / File Transfer)
- **B.png**  — No Local SWIFT Footprint (GUI Access Only)

Supported formats: `.png`, `.jpg`, `.jpeg`, `.svg`, `.webp`

If a file is missing, the card will show a placeholder.
