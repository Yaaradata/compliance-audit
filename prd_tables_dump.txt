=== TABLE 2 (10x2) ===
Decision | Detail
Single Database Engine | PostgreSQL 16 on Google Cloud SQL. No polyglot persistence. JSONB for flexible metadata.
Tenant Isolation | Shared schema + tenant_id + Row-Level Security (RLS). Not schema-per-tenant. RLS policies use current_setting('app.tenant_id', true)::uuid set via API middleware.
External Auth | Authentication handled externally (Firebase Auth / Auth0). No password_hash stored in the DB. users table stores profile, role, and auth_provider_uid only.
Framework Extensibility | audit_frameworks table is framework-agnostic. Phase 2 frameworks (PCI DSS, SOC 2, DORA) add new rows. Cross-framework evidence reuse via shared canonical_evidence_items.
Versioning Strategy | Hybrid: mutable tables + immutable audit_log + phase snapshots. NOT event-sourcing. Current state is the primary query target.
Evidence Scoping | scope_key column handles 4 collection models: standard (__default__), per_system (hostname), per_vendor (vendor UUID), per_quarter (Q1_2025).
File Storage | GCS versioned bucket. Database stores metadata only. Soft-delete only (is_deleted flag). Checksum (SHA-256) tracked.
Audit Log | Partitioned by month (RANGE on timestamp). INSERT-ONLY enforced by database trigger. before_state/after_state as JSONB.
UUID Primary Keys | UUIDs everywhere (uuid_generate_v4). No sequential IDs except audit_log (BIGSERIAL for ordering).
